# SafeZone PlanB

Серверный мод для Project Zomboid (Build 42). Безопасная зона с системой стартовых наборов, радиовещанием и приветственной запиской для новых игроков.

## Возможности

### Starter Kit (Стартовый набор)
- Админ размещает ящик снабжения через контекстное меню
- Игрок кликает по ящику и получает набор экипировки
- Защита от абуза: кулдаун 24 часа, привязка к SteamID
- Состав набора настраивается в конфиге

### Radio (Радиовещание)
- Динамический радиоканал "SafeZone Events" на частоте 95.2 MHz
- Автоматическая трансляция случайного сообщения каждые 10 игровых минут
- Админская команда для отправки сообщений в эфир

### Welcome Note (Приветственная записка)
- Новый игрок при первом заходе получает записку с координатами безопасной зоны
- Выдаётся один раз, дубликаты не создаются

### Кастомная карта
- Собственная карта SafeZone с точкой спавна

## Команды

| Команда | Доступ | Описание |
|---------|--------|----------|
| `/resetkit <ник>` | Админ | Сбросить кулдаун стартового набора игроку |
| `/radio 95.2 <текст>` | Админ | Отправить сообщение в радиоэфир на указанной частоте |

Серверная консоль:
```lua
StarterKitServer.resetKit("ник")
```

## Архитектура

### Клиент-серверная модель

Мод использует асинхронную event-driven коммуникацию PZ: `sendClientCommand` (клиент → сервер) и `sendServerCommand` (сервер → клиент). Вся бизнес-логика и проверки выполняются на сервере — клиент только отправляет запросы и отображает результат.

### Хранение состояния

- **Player ModData** — флаги получения набора и записки (привязаны к персонажу)
- **GameTime ModData** — таблица кулдаунов `StarterKit_Claims[steamID] = timestamp` (привязана к SteamID, переживает смерть персонажа)
- **WorldObject ModData** — маркер `StarterKit_Crate` на объектах-ящиках

### Перехват чат-команд

Кастомные команды (`/resetkit`, `/radio`) реализованы через хук `textEntry.onCommandEntered` на `ISChat.instance`. Хук перехватывает ввод до стандартного обработчика PZ — если команда распознана, она обрабатывается и не передаётся серверу как обычное сообщение.

### Радиосистема

Канал регистрируется через `DynamicRadioChannel` при загрузке радио-скриптов. Трансляции обновляются каждые 10 игровых минут через `Events.EveryTenMinutes`. Команда `/radio` находит канал по частоте в `DynamicRadio.cache` и заменяет текущую трансляцию.

## Конфигурация

Файл `StarterKitConfig.lua`:

```lua
StarterKitConfig = {
    cooldownHours = 24,               -- кулдаун между выдачами (часы)
    crateSprite = "carpentry_01_16",   -- спрайт ящика
    kitItems = {                       -- состав набора
        { item = "Base.Bag_Schoolbag", count = 1 },
        { item = "Base.Hammer", count = 1, container = "bag" },
        -- ...
    },
}
```
